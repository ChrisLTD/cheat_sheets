# Shell Cheat Sheet

## File & Folder Operations

Delete a directory:`
rm -r directory
`

Delete file:`
rm FILETODELETE
`

New directory:`
mkdir FOLDERNAME
`

Copy file:`
cp SOURCE DEST
`

## CURL

Download file:`
curl -O URLHERE
`

## Uzipping

Unzip:`
unzip FILENAME.ZIP
`

Untar & Gzip:`
tar -xzf FILENAME.TAR.GZ
`
